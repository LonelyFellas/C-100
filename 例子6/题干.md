**题目**：用*号输出字母C的图案。
```c
#include <stdio.h>
int main() {
  printf("请你用*打印出字母C: \n");
  printf("    ************\n");
  printf("   ************\n");
  printf("  **\n");
  printf(" **\n");
  printf("**\n");
  printf("**\n");
  printf("**\n");
  printf("**\n");
  printf("**\n");
  printf(" **\n");
  printf("  **\n");
  printf("   ************\n");
  printf("    ************\n");
}
```
**程序分析**：可先用'*'号在纸上写出字母C，再分行输出。
