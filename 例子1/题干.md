**题目**：有 1、2、3、4 四个数字，能组成多少个互不相同且无重复数字的三位数？都是多少？
```c
#include <stdio.h>
int main() {
  int count = 0;
  for(int n = 1; n < 5; n++) {
    for(int i = 1; i < 5; i++) {
      for (int k = 1; k < 5; k++) {
        if (n != i && i != k && k != n) {
          printf("%d,%d,%d\n", n, i, k);
          count++;
        }
      }
    }
  }
  printf("count: %d\n", count);
}
```
**程序分析**：可填在百位、十位、个位的数字都是 1、2、3、4，组成所有的排列后再去掉不满足条件的排列。
